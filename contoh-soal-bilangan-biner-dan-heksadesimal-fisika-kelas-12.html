4. ubah bilangan biner 11100 ke heksadesimal​


Jawaban singkat:

11100 (biner) = 1C (heksadesimal​)

Penjelasan panjang:

Aturan konversi biner (basis 2) ke heksadesimal (basis 16):

- kelompokkan biner jadi 4 digit (dari kanan)

- ubah nilainya dg referensi tabel sistem bilangan.

Sesuai aturan di atas, maka 11100 pengelompokannya menjadi:

0001 (biner) = 1 (heksadesimal)

1100 (biner) = C (heksadesimal)

Jadi 11100 (biner) = 1C (heksadesimal​)

Amatilah tabel sistem bilangan di lampiran.

2. Perhitungan konversi biner ke desimal:

11111 = 1x2⁴+1x2³+1x2²+1x2¹+1x2⁰ = 16 + 8 + 4 + 2 + 1 = 31

Jadi 11111 (biner) = 31 (desimal)

3. Aturan konversi desimal ke heksadesimal:

- bagi bilangan desimal dg angka 16

- tulis hasilnya dan sisa baginya

- penulisan konversi: dari bawah (hasilnya) ke atas (sisa baginya)

Cara perhitungannya bisa dilihat di lampiran (di bawah tabel sistem bilangan).

4. Aturan konversi heksadesimal ke desimal:

- sesuaikan huruf heksadesimal dg angka desimal

- kalikan tiap digit heksadesimal dg angka 16 berpangkat urut

- digit paling kanan dikalikan angka 16 berpangkat 0

- hasil perhitungan di atas, dijumlahkan

Berdasarkan aturan di atas, maka cara perhitungannya yaitu:

2E6 = 2x16² + 14x16¹ + 6x16⁰ = 512 + 224 + 6 = 742

Jadi 2E6 (heksadesimal) = 742 (desimal)

5. Untuk mengkonversi heksadesimal ke biner, dapat dilakukan langsung memakai acuan tabel sistem bilangan (amati lampiran). Aturannnya ialah: 1 digit heksadesimal bernilai 4 digit biner. Jadi

5 (heksadesimal) = 0101 (biner)

E (heksadesimal) = 1110 (biner)

F (heksadesimal) = 1111 (biner)

Jadi 5EF (heksadesimal) = 010111101111 (biner)

010111101111 =0x2¹¹+1x2¹⁰+0x2⁹+1x2⁸+1x2⁷+1x2⁶+1x2⁵+0x2⁴+1x2³+1x2²+1x2¹+1x2⁰

=0+1024+0+256+128+64+32+0+8+4+2+1

=1519

Jadi 010111101111 (biner) = 1519 (desimal)

Sistem bilangan yang sering digunakan adalah bilangan desimal, yaitu bilangan yang terdir dari angka 0, 1, 2, 3, 4, 5, 6, 7, 8 dan 9.

Mengkonversi bilangan biner 1101011 ke dalam bilangan desimal

= 64 + 32 + 0 + 8 + 0 + 2 + 1

= 107

20. Konversikan bilangan biner 11101011011110101101 ke bilangan heksadesimal!


Jawaban:

Bilangan biner 11101011011110101101 setara dengan bilangan heksadesimal EB7AD.

Penjelasan:

Untuk mengonversi bilangan biner (base-2) ke bilangan heksadesimal (base-16), kita perlu membagi bilangan biner menjadi grup empat digit, dimulai dari digit paling kanan. Jika jumlah digit tidak mencapai empat, tambahkan digit 0 di depannya. Setelah itu, konversikan setiap grup empat digit menjadi bilangan heksadesimal.

Bilangan biner 11101011011110101101 dapat dibagi menjadi grup-grup empat digit sebagai berikut: 1110 1011 0111 1010 1101

Kemudian, konversikan setiap grup empat digit menjadi bilangan heksadesimal:

1110 -> E

1011 -> B

0111 -> 7

1010 -> A

1101 -> D

Jadi, bilangan biner 11101011011110101101 setara dengan bilangan heksadesimal E B 7 A D.


21. sama caranya ya kak 1. Konvesikan lah bilangan biner 110110 ke heksadesimal 2.Konvesikanlah bilangan desimal 74 keheksadesimal 3.konversikanlah bilangan heksadesimal 45 ke desimal 4.konversilan lah bilangan heksadesimal 39 ke biner ​


Penjelasan:

1. Untuk mengkonversi bilangan biner 110110 ke heksadesimal, kamu perlu mengelompokkan angka biner menjadi 4-bit, dan kemudian mengkonversinya menjadi nilai heksadesimal yang setara:

110110 -> 00 1101 0110

Sekarang, kita konversi setiap grup 4-bit ke heksadesimal:

00 -> 0 1101 -> D 0110 -> 6

Jadi, bilangan heksadesimalnya adalah 1D6.

2. Untuk mengkonversi bilangan desimal 74 ke heksadesimal, kamu harus membagi bilangan tersebut dengan 16 dan melacak sisanya:

74 / 16 = 4 (sisa 10 -> A)

4 / 16 = 0 (sisa 4)

Karena kita mulai dari bawah ke atas, bilangan heksadesimalnya adalah 4A.

3. Untuk mengkonversi bilangan heksadesimal 45 ke desimal, kamu perlu mengalikan setiap digit dengan pangkat 16 yang sesuai dan menjumlahkannya:

4 * 16^1 + 5 * 16^0 = 64 + 5 = 69

Jadi, bilangan desimalnya adalah 69.

4. Untuk mengkonversi bilangan heksadesimal 39 ke biner, kamu perlu mengkonversi setiap digit menjadi bilangan biner yang setara:

3 -> 0011 9 -> 1001

Dengan menggabungkan kedua bagian ini, kita mendapatkan bilangan biner 00111001.
