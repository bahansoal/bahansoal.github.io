2. Soal no 510 Umptn/b/1995


f(x) = x^3 + 3x^2 + 5 akan turun jika f'(x) < 0
f'(x) = 3x^2 + 6x < 0
=> 3x(x + 2) < 0
x = 0 atau x = -2
+++ (-2) --- (0) +++
-2 < x < 0Kelas : XI (2 SMA)
Materi : Turunan
Kata Kunci : fungsi turun, penyelesaian

Pembahasan :
Diketahui fungsi kuadrat f(x) = x³ + 3x² + 5
Agar fungsi f(x) merupakan fungsi turun, maka f'(x) < 0.
Sehingga
3x² + 6x < 0

Untuk menentukan penyelesaian dari pertidaksamaan kuadrat, maka kita ubah dahulu menjadi persamaan kuadrat
3x² + 6x = 0
⇔ x² + 2x = 0
⇔ x(x + 2) = 0
⇔ x = 0 V x + 2 = 0
⇔ x = 0 V x = -2
Perhatikan gambar terlampir.
Daerah yang negatif merupakan daerah fungsi yang turun, yaitu
-2 < x < 0.

4. Bacalah pembahasan tentang kode genetik!!! Bagaimanakah macam kode genetik terbentuk dan bagaimana cara memahami pesannya?


Kode genetik merupakan kode yang mengolah informasi yang ada di DNA dan RNA menjadi urutan asam amino dan protein.

Kode genetik terdiri dari urutan basa – Sebuah nitrogen, C, G, U – dalam rantai mRNA. Pembentukan kode genetik dimulai dari Empat basa membentuk “huruf” dari kode genetik.

Huruf-huruf tersebut digabungkan dalam tiga kelompok untuk membentuk kode “kata-kata,” disebut kodon. Dan Setiap kodon adalah singkatan (atau mengkodekan) satu asam amino, kecuali kode untuk sinyal memulai atau berhenti.

Umumnya ada 20 asam amino dalam protein. Ada 64 kemungkinan dari kodon, lebih dari cukup untuk kode membuat 20 asam amino. 

Huruf tersebut dibaca dengan bergiliran hingga kode gennya berhenti :

AUGAA

AUG UGA UAG UAA

Volume gas, V = 3000 cm³ = 3.10¯³ m³
Suhu gas, T = 20°C = 293 K
Tekanan alat, P = 25 atm
Tekanan udara luar, Po = 1 atm
Tekanan total, Ptot = 26 atm = 26.10^5 Pa

Massa molar gas oksigen, Mr = 32 kg/kmol = 32 gram/mol
Konstanta gas, R = 8,314 J/mol.K

Tanya :
Massa gas oksigen, m = __?

Jawab :
Step 1
---------
Cari dulu mol nya yak, n = __?
P.V = n.R.T
n = P.V / R.T
n = 26.10^5•3.10¯³ / 8,314•293
n = 7800 / 2436,002
n = 3,2 mol

Step 2
----------
Massa gas oksigen, m = __?
n = m/Mr
m = n.Mr
m = [3,2 mol]•[32 gram/mol]
m = 102,5 gram
m = 0,1025 kg

Atau

m = 0,1 kg
