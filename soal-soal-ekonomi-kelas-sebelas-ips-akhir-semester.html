int (1 sd 2) [√(x-1) /x] dx/x²

= int ( 1 sd 2) √(1-1/x) dx/x²

misal

u = 1-1/x = 1-x^(-1)

du = -(-1) x^(-2) dx

du= dx/x²

int (1 sd 2) √u du

= 2/3 u^(3/2) ( 1 sd 2)

= 2/3 (1-1/x)^(3/2) (1 sd 2)

= 2/3(1/2)^(3/2) -0

= 2/3.1/2 √(1/2)

= 1/3. 1/2√2

= 1/6√2 Jawab E

=int dx/(√x + √(x +1) ) (√x - √x+1)/(√x - √(x +1))

=int (√x - √(x+1)) dx

=int (x^(1/2) -(x+1)^1/2) dx

= 2/3 x^(3/2) - 2/3 (x+1)^(3/2)

= 2/3 [ √x³ - √(x + 1)³ ] + C

y = x² - x - 2
titik potong dengan y = x + 1
x² -x - 2= x  +1
x² -2x - 3= 0
(x - 3)(x  + 1) = 0
x= 3  atau x = -1

Daerah Tertutup pada inteval 0≤ x ≤ 3, 
(diatas garis y  = x +1 , dibawah  y = x² -x - 2)
L =  ₀³ ∫ (x+1)- (x² -x - 2) dx
L =  ₀³∫ x +1 - x² + x + 2 dx
