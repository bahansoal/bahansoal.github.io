7. Soal 1. Carilah IP di masing-masing komputer a. 150 komputer 6.30 komputer C. 10 komputer d. 2 komputer dengan IP AWAL : 200.172.10.10


Jawaban:

Untuk mencari alamat IP pada masing-masing komputer, bisa dilakukan dengan beberapa cara, antara lain:

Pada sistem operasi Windows, buka Command Prompt dan ketikkan "ipconfig" (tanpa tanda petik), kemudian tekan enter. Alamat IP akan terlihat pada bagian "IPv4 Address".

Pada sistem operasi Mac, buka Terminal dan ketikkan "ifconfig" (tanpa tanda petik), kemudian tekan enter. Alamat IP akan terlihat pada bagian "inet".

Setelah mengetahui alamat IP, Anda dapat menggunakan pola alamat IP yang sama dan menambahkan angka pada bagian terakhirnya untuk mencari alamat IP pada komputer lain. Misalnya, jika IP awal adalah 200.172.10.10, maka IP pada komputer berikutnya adalah 200.172.10.11, 200.172.10.12, dst.

Untuk komputer A (150 komputer), bisa ditemukan dengan menambahkan angka dari 0 hingga 149 pada bagian terakhir alamat IP.

Untuk komputer B (6.30 komputer), bisa ditemukan dengan menambahkan angka dari 0 hingga 29 pada bagian terakhir alamat IP.

Untuk komputer C (10 komputer), bisa ditemukan dengan menambahkan angka dari 0 hingga 9 pada bagian terakhir alamat IP.

Untuk komputer D (2 komputer), bisa ditemukan dengan menambahkan angka dari 0 hingga 1 pada bagian terakhir alamat IP.

12. Latihan soal 1. IP (Internet Protocol) Jawablah pertanyaan di bawah ini dengan baik dan benar : 1. Jelaskan pengertian dari IP adress!2. Jelaskan beberapa fungsi dari IP adress!3. Tuliskan dan jelaskan perbedaan dari Network ID dan Host ID!4. Jelaskan cara kerja IP adress dengan bahasamu sendiri! 5. Jelaskan perbedaan antara IPv4 dan IPv6! 6. Jelaskan perbedaan antara IP statis dan IP dinamis!​


Jawaban:

1. IP address (Internet Protocol address) adalah alamat numerik yang unik yang diberikan kepada setiap perangkat yang terhubung ke jaringan komputer yang menggunakan protokol IP. IP address digunakan untuk mengidentifikasi dan berkomunikasi antara perangkat-perangkat tersebut dalam jaringan.

2. Beberapa fungsi dari IP address adalah:

- Identifikasi perangkat: IP address digunakan untuk mengidentifikasi setiap perangkat yang terhubung ke jaringan. Dengan menggunakan IP address, perangkat dapat saling berkomunikasi dan bertukar data.

- Routing data: IP address digunakan dalam proses routing data di jaringan. IP address memungkinkan router untuk mengarahkan data ke tujuan yang tepat melalui jaringan yang kompleks.

- Keamanan jaringan: IP address juga dapat digunakan untuk menerapkan kebijakan keamanan di jaringan. Misalnya, dengan menggunakan IP address, administrator jaringan dapat membatasi akses ke sumber daya jaringan tertentu.

3. Network ID (Network Identifier) dan Host ID (Host Identifier) adalah dua bagian yang terdapat dalam sebuah IP address. Perbedaan antara keduanya adalah:

- Network ID: Bagian dari IP address yang mengidentifikasi jaringan tempat perangkat terhubung. Network ID mengidentifikasi jaringan secara keseluruhan dan dimiliki oleh semua perangkat dalam jaringan yang sama.

- Host ID: Bagian dari IP address yang mengidentifikasi perangkat individu dalam jaringan. Host ID unik untuk setiap perangkat yang terhubung ke jaringan.

4. IP address bekerja dengan menghubungkan perangkat dalam jaringan menggunakan alamat numerik yang unik untuk mengidentifikasi dan berkomunikasi satu sama lain. Ketika perangkat terhubung ke jaringan, ia diberikan IP address yang terdiri dari serangkaian angka. Ketika perangkat ingin mengirim atau menerima data, IP address digunakan untuk mengarahkan data ke perangkat tujuan yang tepat di jaringan. Proses ini melibatkan pemecahan data menjadi paket-paket kecil, dan setiap paket memiliki alamat tujuan yang ditandai dengan IP address.

5. Perbedaan antara IPv4 (Internet Protocol version 4) dan IPv6 (Internet Protocol version 6) adalah:

- IPv4: Menggunakan format IP address 32-bit yang terdiri dari empat blok angka desimal yang dipisahkan oleh titik. IPv4 memiliki jumlah alamat yang terbatas, yaitu sekitar 4,3 miliar alamat. Hal ini menyebabkan kekurangan alamat IP di era modern dengan banyaknya perangkat yang terhubung ke internet.

- IPv6: Menggunakan format IP address 128-bit yang terdiri dari delapan blok huruf dan angka heksadesimal yang dipisahkan oleh titik dua. IPv6 memiliki jumlah alamat yang jauh lebih besar, yaitu sekitar 340 undecillion (3.4 x 10^38) alamat. Hal ini memungkinkan peningkatan jumlah perangkat yang dapat terhubung ke internet dan mengatasi kekurangan alamat IP yang dialami oleh IPv4.

6. Perbedaan antara IP statis dan IP dinamis adalah:

- IP statis: IP statis adalah IP address yang ditetapkan secara manual untuk perangkat dan tidak berubah. IP statis biasanya digunakan untuk perangkat yang membutuhkan alamat yang tetap dan dapat diprediksi, seperti server. IP statis memungkinkan perangkat untuk selalu memiliki alamat yang sama, sehingga lebih mudah untuk mengakses perangkat tersebut.

- IP dinamis: IP dinamis adalah IP address yang diberikan kepada perangkat secara dinamis oleh server DHCP (Dynamic Host Configuration Protocol). IP dinamis dapat berubah setiap kali perangkat terhubung ke jaringan. IP dinamis umumnya digunakan pada perangkat yang tidak memerlukan alamat yang tetap, seperti komputer atau smartphone pengguna biasa. IP dinamis memungkinkan penggunaan alamat IP secara efisien karena alamat dapat dialokasikan dan dikembalikan ke dalam pool alamat yang tersedia.

Jawaban:

Jumlah Subnet dari 192.168.4.5/26 adalah 4

Jumlah Client persubnet 62 host

Jumlah BlokSubnet adalah 64 (1 network address, 1 broadcast, dan 62host)

IP range tiap subnet

192.168.4.1 - 192.168.4.62 untuk subnet 192.168.4.0

192.168.4.65 - 192.168.4.126 untuk subnet 192.168.4.64

192.168.4.129 - 192.168.4.190 untuk subnet 192.168.4.128

192.168.4.193 - 192.168.4.254 untuk subnet 192.168.4.192

Penjelasan:

Jumlah subnet dihitung dengan 2ˣ, x adalah jumlah angka 1 dari oktet terakhir dari subnet masknya, karena subnet mask dgn prefix /26 bernilai 255.255.255.192, angka 192 memiliki binary 11000000 maka 2² = 4

Jumlah Client dapat dihitung dengan 2ⁿ-2, dimana n adalah kebalikan dari x yaitu jumlah 0 pada oktet terakhir yaitu 6, jadi 2⁶-2 = 62

Subnet terdiri dari 192.168.4.0, 192.168.4.64, 192.168.4.128, 192.168.4.192

maka dengan itu IP range (untuk host) dapat diketahui yaitu

192.168.4.1 - 192.168.4.62 dan 192.168.4.63 sbg broadcastnya subnet 192.168.4.0

192.168.4.65 - 192.168.4.126 dan 192.168.4.127 sbg broadcastnya subnet 192.168.4.64

192.168.4.129 - 192.168.4.190 dan 192.168.4.191 sbg broadcastnya subnet 192.168.4.128

192.168.4.193 - 192.168.4.254 dan 192.168.4.255 sbg broadcastnya subnet 192.168.4.192
