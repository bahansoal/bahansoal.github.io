13. 1. nama file = latihan di dokumen di copy ke flashdisk ( drive ) 2. bagaimana mengcopy nama file latihan ke drive ( flashdisk) ke drive D 3. bagaimana mengganti nama file latihan di drive menjadi nama file mencoba 4. bagaimana membuat folder di drive dengan nama teks kelas ​


1. Untuk menyalin file "latihan" dari dokumen ke flashdisk, Anda perlu melakukan langkah-langkah berikut:

- Sambungkan flashdisk ke komputer Anda dan pastikan terdeteksi dan terbaca dengan baik.

- Buka lokasi file "latihan" di dokumen, misalnya di folder "Dokumen".

- Klik kanan pada file "latihan" dan pilih opsi "Salin" atau "Copy".

- Buka flashdisk dengan mengklik ikonnya di komputer Anda.

- Klik kanan di ruang kosong dalam flashdisk dan pilih opsi "Tempel" atau "Paste". File "latihan" akan disalin ke flashdisk.

2. Untuk menyalin nama file "latihan" dari flashdisk ke drive D:

- Buka flashdisk dengan mengklik ikonnya di komputer Anda dan periksa lokasi file "latihan".

- Pastikan flashdisk terdeteksi dengan benar.

- Klik kanan pada file "latihan" dan pilih opsi "Salin Nama File" atau "Copy Nama File".

- Buka drive D, misalnya dengan mengklik ikonnya di komputer Anda.

- Klik kanan di ruang kosong di drive D dan pilih opsi "Tempel" atau "Paste". Nama file "latihan" akan disalin ke drive D.

3. Untuk mengganti nama file "latihan" di drive menjadi "mencoba":

- Buka drive tempat file "latihan" berada, dalam hal ini drive D.

- Cari dan temukan file "latihan".

- Klik kanan pada file "latihan" dan pilih opsi "Ubah Nama" atau "Rename".

- Ketikkan nama baru "mencoba" dan tekan Enter. Nama file "latihan" akan diganti menjadi "mencoba".

4. Untuk membuat folder di drive dengan nama "teks kelas":

- Buka drive tempat Anda ingin membuat folder baru, misalnya drive D.

- Klik kanan di ruang kosong di drive D dan pilih opsi "Buat Folder" atau "Create New Folder".

- Ketikkan nama folder baru "teks kelas" dan tekan Enter. Folder baru dengan nama "teks kelas" akan dibuat di drive D.

26. langkah untuk menformat dokumen LATIH-2 dengan ukuran kertas A4 dengan batas atas dan kiri 3 cm batas bawah dan kanan 2 cm dan dokumen dikompres menjadi 75%

1. Klik Page Layout
2. Klik Size
3. Klik A4

Cara mengatur Margins:

1. Klik Page Layout
2. Klik Margins
3. Custom Margins
4. Ubah menjadi:
     - Top: 3cm
     - Left: 3cm
     - Bottom: 2cm
     - Right: 2cm

Untuk mengubah nama dokumen:

1. Klik Office Button
2. Klik Save as
3. Namai dokumen dengan "LATIH-2"

Cara memperkecil ukuran dokumen:

1. Klik salah satu gambar yang terdapat pada dokumen
2. Pada tab Format pilih Size
3. Maka akan tampil windows baru, lalu pilih tab picture, setelah itu klik tombol compress.
4. Setelah mengklik tombol compress maka akan tampil windows baru, lalu pastikan option nya sama dengan gambar di bawah
5. Maka akan muncul konfimasi, tekan saja Apply untuk melajutkan proses
6. Jika sudah lakukan proses save as pada file dokumen tersebut dan lihat hasilnya.
