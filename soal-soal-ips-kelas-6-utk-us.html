13. tolong dijawab soalnya di scroll kanan dan kiri utk melihat soalnya


1. - tidak terlalu mementingkan tampilan grafis
- proses loading cepat
- membutuhkan spesifikasi hardware yang rendah

2. Pada Windows 7:
- Switch User: untuk pergantian pengguna komputer tanpa menutup program.
- Log off: untuk pergantian pengguna komputer dengan menutup program.
- Lock: untuk mengunci komputer sehingga apabila ingin menggunakan komputer kembali, pengguna harus memasukan passwordnya.
- Restart: untuk menutup semua program yang sedang berjalan dan menon-aktifkan komputer sementara kemudian secara otomatis komputer akan hidup kembali.
- Sleep: fungsinya hampir sama dengan Lock bedanya jika kita mengaktif kan fungsi sleep maka komputer berada dalam mode penghematan power dan layar komputer akan berubah menjadi gelap/mati.
- Shut down: untuk mematikan komputer dan menutup semua program yang sedang berjalan.
- Hibernate: untuk mematikan komputer tanpa menutup semua program yang sedang berjalan.

3. Masuk ke dalam control panel Windows 7:
- klik User Accounts and family safety
- klik Add or remove user accounts
- klik Create a new account
- isikan nama akun dan pilih tipe akun (standard user atau Administrator)
- klik tombol Create Account

4. Memakai keyboard: tekan tombol Ctrl+N
memakai mouse: klik tombol New pada quick access toolbar
cara lainnya: klik tab. File > New > Blank Document > Create

5. Klik ribbon Page Layout > klik tombol Page Setup > pilihlah ukuran kertas di menu Paper Size > klik tombol OK
