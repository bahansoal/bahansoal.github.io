2sin (2x + phi) = V3
sin (2x + phi) = (V3)/2
sin (2x + phi) = 60°
sin (2x + phi) = sin 60°
2x + phi = 60° + k.360
2x = 60 - phi + k.360
2x = -120 + k.360
x = -60 + k.180
untuk k = 0, => x = -60
untuk k = 1, => x = 120
untuk k = 2 => x = 300
