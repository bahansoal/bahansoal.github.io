2. Jika fungsi kuadrat f(x) = (a – 3)x2 + ax + 4 mempunyai nilai ekstrem (a + 3), maka nilai ekstrem fungsi ini adalah


Jika fungsi kuadrat f(x) = (a – 3)x2 + ax + 4 mempunyai nilai ekstrem (a + 3), maka nilai ekstrem fungsi ini adalah ada 2 yaitu [tex]y_e=21/5[/tex] dan [tex]y_e=5[/tex]

Pembahasan:

ada beberapa cara untuk mencari nilai ekstrem suatu fungsi yaitu

(i)mencari terlebih dahulu sumbu simetrinya [tex]x_s=\frac{-B}{2A}[/tex] kemudian mensubtitusikanya ke fungsi [tex]f(x)[/tex] tersebut

(ii) menggunakan rumus [tex]y_e=\frac{B^{2}-4AC}{-4A}[/tex]

dengan :

[tex]A=(a-b)[/tex]

[tex]B=a[/tex]

[tex]C=4[/tex]

disini saya akan menggunakan cara (ii)

Diketahui:

[tex]f(x)=(a-3)x^{2}+ax+4[/tex]

[tex]y_e=a+3[/tex]

Ditanya:

[tex]y_e=...[/tex]

Penyelesaian:

[tex]y_e=\frac{B^{2}-4AC}{-4A}[/tex]

[tex]=\frac{a^{2}-4(a-3)a}{-4(a-3)}[/tex]

[tex]=\frac{a^{2}-16a+48}{-4(a-3)}[/tex]

[tex]a+3=\frac{a^{2}-16a+48}{-4(a-3)}[/tex]

[tex]\frac{-4(a-3)(a+3)}{-4(a-3)}=\frac{a^{2}-16a+48}{-4(a-3)}[/tex]

[tex]\frac{a^{2}-16a+48+4a^{2}-36}{-4(a-3)}=0[/tex]

[tex]a^{2}-16a+48+4a^{2}-36=0[/tex]

[tex]5a^{2}-16a+12=0[/tex]

[tex](5a-6)(a-2)=0[/tex]

didapat

[tex]a=6/5[/tex]  dan  [tex]a=2[/tex]

jadi untuk  [tex]a=6/5[/tex] maka nilai ekstremnya yaitu [tex]y_e=6/5+3=21/5[/tex]

untuk  [tex]a=2[/tex] maka [tex]y_e=2+3=5[/tex]

3. diketahui fungsi kuadrat F(x)=2x²-5x-2 tentukanlah nilai ekstrem dan titik ekstremnya​


Jawaban:

Untuk menentukan nilai ekstrem dan titik ekstrem dari fungsi kuadrat F(x) = 2x² - 5x - 2, kita perlu menggunakan rumus:

x = -b/2a

di mana a, b, dan c adalah koefisien dari persamaan kuadratik ax² + bx + c = 0. Dalam hal ini, a = 2 dan b = -5.

Untuk menentukan tanda koefisien a agar mudah menentukan apakah titik ekstrimnya merupakan maksimum atau minimum, kita bisa melihat tanda koefisien a. Karena a > 0, maka fungsi ini memiliki parabola yang membuka ke atas, sehingga titik ekstremnya merupakan titik minimum.

Dengan menggunakan rumus di atas, kita dapat menghitung nilai x dari titik ekstrem:

x = -(-5)/2(2)

x = 5/4

Untuk menentukan nilai y dari titik ekstrem, kita bisa memasukkan nilai x ke dalam fungsi F(x):

F(x) = 2x² - 5x - 2

F(5/4) = 2(5/4)² - 5(5/4) - 2

F(5/4) = 2(25/16) - 25/4 - 2

F(5/4) = 25/8 - 25/4 - 2

F(5/4) = -57/8

Maka, nilai ekstrem dari fungsi F(x) adalah -57/8, dan titik ekstremnya terletak pada x = 5/4, yaitu (5/4, -57/8).

4. carilah titik potong ekstrem ( titik potong) , nilai ekstrem dan buatlah sketsa grafiknya dari fungsi kuadrat berikut ( b) y+×²=4×-5​


Jawaban:

Untuk mencari titik potong ekstrem dan nilai ekstrem dari fungsi kuadrat y = ax^2 + bx + c, kita perlu mengetahui nilai koefisien a, b, dan c. Dalam kasus ini, fungsi kuadrat yang diberikan adalah y = x^2 + 4x - 5.

1. Mencari titik potong dengan sumbu x:

Untuk mencari titik potong dengan sumbu x, kita perlu mencari nilai x ketika y = 0 dalam fungsi kuadrat tersebut.

0 = x^2 + 4x - 5

Kita dapat menggunakan metode faktorisasi atau menggunakan rumus kuadrat untuk menyelesaikan persamaan ini. Dalam hal ini, kita akan menggunakan rumus kuadrat:

x = (-b ± √(b^2 - 4ac)) / 2a

a = 1

b = 4

c = -5

x = (-(4) ± √((4)^2 - 4(1)(-5))) / 2(1)

x = (-4 ± √(16 + 20)) / 2

x = (-4 ± √36) / 2

x = (-4 ± 6) / 2

x1 = (-4 + 6) / 2

x1 = 2 / 2

x1 = 1

x2 = (-4 - 6) / 2

x2 = -10 / 2

x2 = -5

Jadi, titik potong dengan sumbu x adalah x = 1 dan x = -5.

2. Mencari nilai ekstrem (titik potong dengan sumbu y):

Untuk mencari nilai ekstrem, kita perlu mencari nilai y ketika x = -b/2a.

x = -4 / (2 * 1)

x = -2

y = (-2)^2 + 4(-2) - 5

y = 4 - 8 - 5

y = -9

Jadi, nilai ekstrem atau titik potong dengan sumbu y adalah y = -9.

3. Sketsa grafik fungsi kuadrat:

Dengan informasi yang diberikan di atas, kita dapat membuat sketsa grafik fungsi kuadrat y = x^2 + 4x - 5.

Titik potong dengan sumbu x: (1, 0) dan (-5, 0)

Titik potong dengan sumbu y: (0, -9)

Grafik fungsi kuadrat y = x^2 + 4x - 5 akan menunjukkan parabola yang terbuka ke atas, dan dengan titik potong pada (1, 0), (-5, 0), dan (0, -9).

9. Diketahui nilai ekstrem fungsi f(x)=-x²+mx-m adalah 15. Nilai m adalah


Nilai m yang memenuhi fungsi kuadrat f(x) = -x² + mx - m adalah m = -6 atau m = 10. Nilai tersebut diperoleh dari perhitungan nilai ekstrim fungsi kuadrat f(x) = -x² + mx - m. SImak pembahasan berikut.

Pembahasan

Diketahui:

Fungsi kuadrat f(x) = -x² + mx - m

Nilai ekstrim fungsi f(x) = 15

Ditanya: Nilai m

Jawab:

Nilai ekstrim suatu fungsi kuadrat f(x) = ax² + bx + c dirumuskan dengan

y = -D/4a

dengan D = b² - 4ac

Maka nilai ekstrim fungsi f(x) = -x² + mx - m diperoleh sebagai berikut:

f(x) = -x² + mx - m

a = -1, b = m, c = -m

D = b² - 4ac

D = m² - 4(-1)(-m)

D = m² - 4m

y = -D/4a

15 = -(m² - 4m)/4(-1)

15 = -(m² - 4m)/(-4)

15 = (m² - 4m)/4

15 × 4 = m² - 4m

60 = m² - 4m

m² - 4m - 60 = 0

Menentukan nilai m dengan menggunakan metode melengkapi kuadrat sempurna

m² - 4m - 60 = 0

m² - 4m = 60

m² - 4m + (1/2 × (-4))² = 60 + (1/2 × (-4))²

m² - 4m + (-2)² = 60 + (-2)²

m² - 4m + (-2)² = 60 + 4

(m - 2)² = 64

m - 2 = ± √64

m - 2 = ± 8

m - 2 = 8

m = 8 + 2

m = 10

atau

m - 2 = -8

m = -8 + 2

m = -6

Maka nilai m yang memenuhi adalah m = -6 atau m = 10.

16. Jika diketahui fungsi produksi perusahaan meubel z (X,Y) = 2x2 y2. Indentifikasi titik ekstrem fungsi produksi perusahaan tersebut. Berapa jumlah barang X dan Y yang harus diproduksi dan hitung nilai ekstremnya.


Jawaban:

Untuk mengidentifikasi titik ekstrem fungsi produksi z(X, Y) = 2X^2Y^2, kita perlu mencari nilai minimum atau maksimum dengan mengambil turunan parsial terhadap variabel X dan Y.

Turunan parsial terhadap X:

∂z/∂X = 4XY^2

Turunan parsial terhadap Y:

∂z/∂Y = 4X^2Y

Untuk mencari titik ekstrem, kita set turunan parsial terhadap X dan Y menjadi nol:

4XY^2 = 0

4X^2Y = 0

Dari persamaan di atas, kita bisa mendapatkan dua kemungkinan titik ekstrem:

1. Ketika X = 0 dan Y ≠ 0

Dalam hal ini, jumlah barang X yang harus diproduksi adalah 0, dan kita dapat memilih jumlah barang Y yang tidak sama dengan 0. Namun, nilai ekstrem tidak dapat ditentukan karena Y dapat memiliki berbagai nilai.

2. Ketika X ≠ 0 dan Y = 0

Dalam hal ini, jumlah barang Y yang harus diproduksi adalah 0, dan kita dapat memilih jumlah barang X yang tidak sama dengan 0. Nilai ekstrem akan bergantung pada nilai X yang dipilih.

Jadi, tidak ada jumlah barang X dan Y yang dapat ditentukan dengan pasti untuk mencapai titik ekstrem. Namun, jika kita memilih nilai X ≠ 0 dan Y = 0 (atau sebaliknya), kita dapat menghitung nilai ekstrem sesuai dengan nilai X yang dipilih.
