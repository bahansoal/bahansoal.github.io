18. bantuin soal matematika kelas XI


f(x) = (2x + 2)/(x + 8)
y = (2x + 2)/(x + 8)
y(x + 8) = 2x + 2
xy + 8y = 2x + 2
xy - 2x = 2 - 8y
x(y - 2) = 2 - 8y
x = (2 - 8y)/(y - 2)
f⁻¹(x) = (2 - 8x)/(x - 2)

TRIGONOMETRI

Jawab :

• rumus identitas:

tan x = sin x/ cos x

sin² x + cos² x = 1

sec x = 1/cos x

sin x. tan x + cos x = sec x

(sin² x/cos x) + cos x = sec x

(sin² x + cos² x)/cos x = sec x

1/cos x = sec x

sec x = sec x.
