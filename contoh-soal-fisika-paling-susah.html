Jawaban:

Sebuah peluru ditembakkan dari tanah dengan sudut elevasi 45° dan kecepatan awal 20 m/s. Hitung panjang lintasan dari posisi awal hingga mendarat di tanah!

Penjelasan:

Gerak parabola merupakan GLB di sumbu X dan GLBB di sumbu Y. Maka persamaan posisi peluru

[tex]\displaystyle x=v_xt=v_0\cos\theta~t\\y=v_yt-\frac{1}{2}gt^2=v_0\sin\theta~t-\frac{1}{2}gt^2[/tex]

Sekarang cari fungsi parabola nya

[tex]\displaystyle x=v_0\cos\theta~t\rightarrow t=\frac{x}{v_0\cos\theta}[/tex]

maka

[tex]\begin{aligned}y&\:=v_0\sin\theta~t-\frac{1}{2}gt^2\\\:&=v_0\sin\theta\left ( \frac{x}{v_0\cos\theta} \right )-\frac{1}{2}g\left ( \frac{x}{v_0\cos\theta} \right )^2\\\:&=x\tan\theta-\frac{1}{2}g\left ( \frac{x}{v_0\cos\theta} \right )^2\\\:&=x\tan 45^\circ-\frac{1}{2}(10)\left ( \frac{x}{20\cos 45^\circ} \right )^2\\\:&=x-\frac{1}{40}x^2\end{aligned}[/tex]

Apabila di tanah maka y = 0 sehingga

[tex]\displaystyle y=x-\frac{1}{40}x^2\\0=x-\frac{1}{40}x^2\\0=x\left ( 1-\frac{1}{40}x \right )\\x=0~\mathrm{m}~\vee~x=40~\mathrm{m}[/tex]

Dengan tinjauan diferensial untuk menentukan panjang lintasan

[tex]\displaystyle ds^2=dx^2+dy^2\\\frac{ds^2}{dx^2}=1+\frac{dy^2}{dx^2}\\\frac{ds}{dx}=\sqrt{1+\left ( \frac{dy}{dx} \right )^2}\\\boxed{s=\int_{a}^b \sqrt{1+y'^2}~dx}[/tex]

Rumus diatas adalah rumus panjang busur. Berlaku untuk semua jenis kurva.

[tex]\displaystyle y=x-\frac{1}{40}x^2\rightarrow y'=1-\frac{1}{20}x[/tex]

Agar mempermudah perhitungan dimisialkan

[tex]\displaystyle u=1-\frac{1}{20}x\rightarrow du=-\frac{1}{20}~dx[/tex]

Sekarang cari batas atas dan bawah pengintegralan

[tex]\displaystyle x=0\rightarrow u=1-\frac{1}{20}(0)=1\\x=40\rightarrow u=1-\frac{1}{20}(40)=-1[/tex]

Batas bawah 1 dan batas atas -1

[tex]\begin{aligned}s&\:=\int_{a}^{b}\sqrt{1+\left ( 1+\frac{1}{20}x \right )^2}~dx\\\:&=-20\int_{1}^{-1}\sqrt{1+u^2}~du\end{aligned}[/tex]

Pengintegralan ini melibatkan substitusi trigonometri atau hiperbolik. Saya gunakan substitusi trigonometri

[tex]\begin{array}{|c|c|}\cline{1-2} \rm{Bentuk} & \rm{Substitusi}\\\cline{1-2} \sqrt{a^2-u^2} & u=a\sin\theta\\\cline{1-2} \sqrt{a^2+u^2} & u=a\tan\theta\\\cline{1-2} \sqrt{u^2-a^2} & u=a\sec\theta\\\cline{1-2}\end{array}[/tex]

Operasikan [tex]\displaystyle \int \sqrt{1+u^2}~du[/tex]

[tex]\displaystyle u=\tan\theta\rightarrow du=\sec^2\theta~d\theta[/tex]

[tex]\begin{aligned}\int \sqrt{1+u^2}~du&\:=\int \sqrt{1+\tan^2\theta}~\sec^2\theta~d\theta\\\:&=\int \sec^3\theta~d\theta\end{aligned}[/tex]

Cari [tex]\displaystyle \int \sec^3\theta~d\theta[/tex] dengan mengubah nya menjadi [tex]\displaystyle \int \sec\theta\sec^2\theta~d\theta[/tex] gunakan integral parsial.

[tex]\displaystyle \begin{matrix}u=\sec\theta & dv=\sec^2\theta~d\theta\\ du=\tan\theta\sec\theta~d\theta & v=\tan\theta\end{matrix}[/tex]

[tex]\displaystyle \int \sec^3\theta~d\theta=\int \sec\theta\sec^2\theta~d\theta\\\int \sec^3\theta~d\theta=\sec\theta\tan\theta-\int \tan^2\theta\sec\theta~d\theta\\\int \sec^3\theta~d\theta=\sec\theta\tan\theta-\int (\sec^2\theta-1)\sec\theta~d\theta\\\int \sec^3\theta~d\theta=\sec\theta\tan\theta-\int \sec^3\theta~d\theta-\int \sec\theta~d\theta\\2\int \sec^3\theta=\sec\theta\tan\theta+\int \sec\theta~d\theta[/tex]

Cari [tex]\displaystyle \int \sec\theta~d\theta[/tex]

[tex]\begin{aligned}\int \sec\theta~d\theta&\:=\int \frac{\sec\theta(\sec\theta+\tan\theta)}{\sec\theta+\tan\theta}~d\theta\\\:&=\int \frac{\sec^2\theta+\sec\theta\tan\theta}{\sec\theta+\tan\theta}~d\theta\\\:&=\int \frac{\sec^2\theta+\sec\theta\tan\theta}{u}~\frac{du}{\sec\theta\tan\theta+\sec^2\theta}\\\:&=\ln|u|+C\\\:&=\ln|\sec\theta+\tan\theta|+C\end{aligned}[/tex]

maka

[tex]\displaystyle \int \sec^3\theta~d\theta=\frac{\sec\theta\tan\theta+\ln|\sec\theta+\tan\theta|}{2}+C[/tex]

sehingga

[tex]\begin{aligned}s&\:=-20\int_{1}^{-1} \sqrt{1+u^2}~du\\\:&=-20\left [ \frac{\tan\theta\sec\theta+\ln|\tan\theta+\sec\theta|}{2} \right ]_1^{-1}\\\:&=-20\left [ \frac{u\sqrt{1+u^2}+\ln|u+\sqrt{1+u^2}|}{2} \right ]_1^{-1}\\\:&=-20\left [ \frac{u\sqrt{1+u^2}+\sinh^{-1}u}{2} \right ]_1^{-1}\\\:&\approx -20(-1,147-1,147)\\\:&\approx -45,88~\mathrm{m}\end{aligned}[/tex]

GAYA COULOMB

Gaya Coulomb dinyatakan dengan persamaan

[tex]F \:=\: k \frac{Q_1 \: Q_2}{r^2}[/tex]

dimana

k = [tex]9 \times 10^9[/tex] Nm²/C²Q = muatan (C)r = jarak (m)F = gaya Coulomb (N)

31. kak bantuin donk fisika no 3 sma 4 susah


Nomor 3
F = kq² / r²
q² = Fr² / k
q = √(Fr² / k)
    = √[(6,4 N)(1,5 m)² / (9 x 10⁹ Nm²/C²)]
    = √[(14,4 Nm²) / (9 x 10⁹ Nm²/C²)]
    = √[(1,6 x 10⁻⁹ C²)]
    = √[(16 x 10⁻¹⁰ C²)]
    = 4 x 10⁻⁵ C

Nomor 4
F / F' = (kq₁q₂ / r²) / (kq₁q₂ / r'²)
F / F' = (r' / r)²
F / F' = (2r / r)²
F / F' = 4 / 1
F' = 1/4 F

200 N

Penjelasan:

Hukum Pascal : F1/A1 = F2/A2

F1 (yang dicari); F2 = yang di penampang besar

A = Luas permukaan (satuannya biarkan saja, karena nanti akan dicoret di kedua ruas)

F1/50 = 16000/4000, maka F1 = 4*50 = 200 N

diket = a1 : 50 cm^2

a2 : 4000 cm^2

F2 : 16.000 N

ditanya = F1

jawab

F1 = A1 /A2 × F2

= 50 × 10^-4/ 4.000 × 10 ^-4 × 16.000 N

= 0.0125 × 16.000

= 200 N
