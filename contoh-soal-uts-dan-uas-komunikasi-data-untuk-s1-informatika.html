12. Soal quiz Konversikan Pseudo code algoritma nilai ke dalam flowchart? 1. DEKLARASI2. tugas, uas, uts, rata_rata_nilai: real 3. DESKRIPSI4.read(tugas, uts, uas) 5. rata_rata_nilai = (tugas+uas+uts)/3 6. Write(rata_rata_nilai)​


Pseudo code di atas bisa kita konversi menjadi flowchart (diagram alur) yang menunjukkan bagaimana proses pemrograman yang terjadi. Flowchart tersebut bisa dilihat di dalam lampiran gambar di bawah.

Pembahasan

Di dalam pseudo code di atas kita lihat ada 4 proses utama yang terjadi, yaitu:

1. Proses deklarasi variable yang dibutuhkan beserta tipe datanya.

2. Input data yang diperlukan untuk disimpan di dalam variable yang sudah kita tentukan.

3. Proses penghitungan data berdasarkan formula yang diberikan di dalam deklarasi, yaitu:

rata_rata_nilai = (tugas+uas+uts)/3

4. Proses output yaitu berupa print hasil formula di atas ke dalam console.

Pseudo code di atas setelah kita konversikan ke dalam flowchart lalu kita ubah ke dalam bahasa pemrograman sesuai pilihan dan kebutuhan kita, misalnya Python, Java, C++, dan lainnya.

1.  Buatlah sebuah program yang dapat mengolah data mahasiswa dengan ketentuan:

User menginputkan NIM, Nama, Nilai Kuis, Nilai UTS, dan Nilai UAS. Hasil atau output yang diinginkan: NIM, Nama, Nilai Kuis, Nilai UTS, Nilai UAS, dan Nilai Akhir. Nilai akhir = 20% dari nilai kuis + 30% dari nilai UTS + 50% dari nilai UAS.

JAWABANNYA :

1. program recordmhs;

uses crt ;

type

mahasiswa = record

nim : real;

nama : string[30] ;

kuis: real;

uts: real;

uas: real;

akhir: real;

end;

var

recMhs : array[1..50]of mahasiswa ;

byk: byte;

i,j,k: integer;

begin

clrscr;

write('Masukkan jumlah mahasiswa: '); readln(byk);

for i:= 1 to byk do

begin

write('NIM : '); readln(recMhs[i].nim) ;

write('Nama : '); readln(recMhs[i].nama) ;

write('Kuis: '); readln(recMhs[i].kuis) ;

write('UTS: '); readln(recMhs[i].uts) ;

write('UAS: '); readln(recMhs[i].uas) ;

writeln;

end;

writeln;

for i:= 1 to byk do

begin

recMhs[i].akhir:= 0.2* recMhs[i].kuis + 0.3 * recMhs[i].uts + 0.5 * recMhs[i].uas ;

writeln('NIM : ',recMhs[i].nim:0:1) ;

writeln('Nama : ',recMhs[i].nama) ;

writeln('Kuis: ',recMhs[i].kuis:0:1) ;

writeln('UTS: ',recMhs[i].uts:0:1) ;

writeln('UAS: ',recMhs[i].uas:0:1) ;

writeln('Akhir: ',recMhs[i].akhir:0:1);

writeln;

end;

readln;

end.

33. contoh penerapan informatika dalam kehidupan sehari-hari selain (informasi, komunikasi, entertaint, dan edukasi).​


Jawaban:

Selain informasi, komunikasi, hiburan, dan edukasi, penerapan informatika juga dapat ditemukan dalam berbagai aspek kehidupan sehari-hari. Berikut adalah beberapa contoh:

Kesehatan dan Kesejahteraan:

Informatika digunakan dalam pengembangan aplikasi kesehatan, seperti pelacakan aktivitas fisik melalui perangkat wearable, monitoring kesehatan melalui aplikasi mobile, dan pengelolaan jadwal perawatan medis.

Transportasi dan Mobilitas:

Aplikasi peta digital, layanan ridesharing, dan sistem navigasi GPS adalah contoh penerapan informatika dalam mempermudah perjalanan dan mobilitas sehari-hari.

Keuangan dan Pembayaran:

Informatika berperan dalam pengelolaan keuangan pribadi, aplikasi perbankan digital, dan layanan pembayaran online yang memudahkan transaksi sehari-hari.

Pengelolaan Rumah Tangga:

Aplikasi untuk mengatur jadwal, membeli bahan makanan secara online, serta mengontrol perangkat rumah pintar (smart home) menggunakan teknologi informatika.

Belanja dan E-Commerce:

E-commerce dan aplikasi belanja online memanfaatkan informatika untuk memungkinkan pembelian produk dan layanan dengan mudah melalui platform digital.

Pengelolaan Waktu:

Aplikasi pengingat, kalender digital, dan alat manajemen waktu memanfaatkan informatika untuk membantu pengguna mengatur jadwal dan aktivitas sehari-hari.

Transportasi Umum:

Sistem tiket elektronik dan informasi perjalanan real-time dalam transportasi umum merupakan penerapan informatika untuk mempermudah penggunaan layanan transportasi.

Pengaturan Energi:

Teknologi smart metering dan aplikasi pengaturan energi memungkinkan pengguna mengontrol konsumsi energi di rumah atau tempat kerja.

Pertanian dan Agribisnis:

Informatika digunakan dalam pemantauan pertanian melalui sensor-sensor, aplikasi prediksi cuaca, dan teknologi pemetaan lahan pertanian.

Logistik dan Pengiriman:

Sistem pelacakan pengiriman barang, pengaturan rute pengiriman, serta manajemen inventaris menggunakan teknologi informatika.

Pariwisata dan Perjalanan:

Aplikasi untuk merencanakan perjalanan, reservasi akomodasi, dan eksplorasi wisata menggunakan teknologi informatika.

Penerapan informatika yang luas mencakup hampir semua aspek kehidupan sehari-hari, menghadirkan kenyamanan dan efisiensi dalam berbagai aktivitas dan interaksi kita.
