=> 1/Rp = 1/6 + 1/12
=> 1/Rp = 2/12 + 1/12
=> 1/Rp = 3/12 = 1/4
=> Rp = 4 ohm

Rtotal = Rs + Rp
=> Rt = 6 + 4
=> Rt = 10

Itotal = V/Rt
=> It = 6/10
=> It = 0,6 A

I(L1) = 0,6 A

perbandingan (L3+L4) & L2 = (4+2) : 12
=> 6 : 12 = 1 : 2
R(L3 & L4)/R(L2) = I(L2)/I(L3 & L4)
=> 1/2 = I(L2)/I(L3 & L4)

I(L2) = 1/(1+2) × 0,6

I(L3 & L4) = 2/(1+2) × 0,6
