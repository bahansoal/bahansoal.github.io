33. jika x²+y²+xy=200 dan xy+x+y=72 maka berapakah (x-y)²


Teori Bilangan

xy + x + y = 72

xy = 72 - (x + y)

x² + y² + xy = 200

x² + y² + 72 - (x + y) = 200

(x + y)² - 2xy - (x + y) = 128

(x + y)² - 2(72 - (x + y)) - (x + y) = 128

(x + y)² + (x + y) - 144 - 128 = 0

((x + y) - 16)((x + y) + 17) = 0

x + y = 16 atau x + y = -17

xy + x + y = 72

xy = 72 - (x + y)

xy = 72 - 16

xy = 56

(x - y)²

= x² + y² - 2xy

= x² + y² + xy - 3xy

= 200 - 3xy

= 200 - 3(56)

= 200 - 168

= 32 ✔

40. Diketahui segitiga XYZ siku-siku di Y, Jika XY = 3cm dan YZ = 4cm . hitunglah nilai Sin X + Tan X , dengan X merupakan sudut antara XY dan YZ ! tolong dijawab ya,buat besok soalnya ;") terimakasih.


Nilai dari sin x + tan x = [tex]\frac{4}{5} +\frac{4}{3}[/tex] = [tex]\frac{4*3+4*5}{5*3}[/tex] = [tex]\frac{12+20}{15}[/tex] = [tex]\frac{32}{15}[/tex]. Unttuk menyelesaikan soalini,kita gunakan pythagoras.

Penjelasan dengan langkah-langkah:

Diketahui:

Segitiga XYZ siku-siku di Y,

Jika XY = 3cm dan YZ = 4cm .

Ditanya:

Hitunglah nilai Sin X + Tan X , dengan X merupakan sudut antara XY dan YZ !

Jawab:

Perhatikan gambar pada lampiran.

Dengan menggunakan teorema Pythagoras, diperoleh

XY² + ZY² = XZ²

XZ² = 3² + 4²

XZ² = 9 + 16

XZ² = 25

XZ = √25

XZ = 5

Kemudian,

sin X =  [tex]\frac{YZ}{XZ}[/tex]

sin X =  [tex]\frac{4}{5}[/tex]

tan X =  [tex]\frac{YZ}{YX}[/tex]

tan X = [tex]\frac{4}{3}[/tex]

cos X =  [tex]\frac{YZ}{XZ}[/tex]
