5. diketahui ip address 192.168.10.100 tuliskan a. jumlah subnet b. ip validc. ip broadcast​


Untuk menjawab pertanyaan Anda, diperlukan informasi tambahan tentang subnet mask yang digunakan dalam jaringan tersebut. Subnet mask diperlukan untuk mengidentifikasi subnet, alamat IP yang valid, dan alamat IP broadcast.

Misalkan kita anggap subnet mask yang digunakan adalah 255.255.255.0 (CIDR: /24), berikut adalah jawaban untuk setiap pertanyaan:

a. Jumlah Subnet:

Jumlah subnet dapat ditentukan dengan melihat jumlah bit yang digunakan untuk subnet pada subnet mask. Dalam kasus ini, subnet mask adalah 255.255.255.0, yang berarti ada 24 bit yang digunakan untuk subnet dan 8 bit yang digunakan untuk host. Dengan 24 bit subnet, jumlah subnet yang mungkin adalah 2^24 - 2 (2^24 mengurangi subnet yang terreservasi untuk alamat jaringan dan alamat broadcast). Jadi, jumlah subnet yang mungkin adalah 2^24 - 2 = 16.777.214 subnet.

b. Alamat IP Valid:

Alamat IP yang valid dalam subnet ini adalah alamat yang digunakan untuk host. Dalam kasus ini, alamat IP 192.168.10.100 adalah alamat host, jadi itu adalah alamat IP yang valid dalam subnet tersebut.

c. Alamat IP Broadcast:

Alamat IP broadcast adalah alamat yang digunakan untuk mengirim pesan ke semua host dalam subnet. Dalam kasus ini, alamat IP broadcast dapat ditentukan dengan mengatur semua bit host pada alamat IP menjadi 1. Jadi, alamat IP broadcast untuk subnet tersebut adalah 192.168.10.255.

9. tentukan ip address 192.168.10.100/30 a) ip network b)ip broadcast c) ip host d)sub netmask​


Jawaban:

Untuk alamat IP 192.168.10.100/30:

a) IP Network: 192.168.10.100/30 memiliki alamat IP jaringan 192.168.10.100.

Pada subnet /30, dua bit terakhir pada oktet keempat (100) digunakan untuk alamat host, sehingga alamat IP jaringannya adalah 192.168.10.100.

b) IP Broadcast: IP broadcast pada subnet /30 adalah alamat IP terakhir dalam jaringan. Dalam hal ini, alamat IP broadcast adalah 192.168.10.103.

c) IP Host: IP host dalam subnet /30 adalah alamat IP di antara IP jaringan dan IP broadcast. Dalam hal ini, alamat IP host yang valid adalah 192.168.10.101 dan 192.168.10.102.

d) Subnet Mask: Subnet mask untuk /30 adalah 255.255.255.252. Subnet mask ini digunakan untuk membagi alamat IP menjadi alamat jaringan dan alamat host.

Jawaban:

Jumlah Subnet dari 192.168.4.5/26 adalah 4

Jumlah Client persubnet 62 host

Jumlah BlokSubnet adalah 64 (1 network address, 1 broadcast, dan 62host)

IP range tiap subnet

192.168.4.1 - 192.168.4.62 untuk subnet 192.168.4.0

192.168.4.65 - 192.168.4.126 untuk subnet 192.168.4.64

192.168.4.129 - 192.168.4.190 untuk subnet 192.168.4.128

192.168.4.193 - 192.168.4.254 untuk subnet 192.168.4.192

Penjelasan:

Jumlah subnet dihitung dengan 2ˣ, x adalah jumlah angka 1 dari oktet terakhir dari subnet masknya, karena subnet mask dgn prefix /26 bernilai 255.255.255.192, angka 192 memiliki binary 11000000 maka 2² = 4

Jumlah Client dapat dihitung dengan 2ⁿ-2, dimana n adalah kebalikan dari x yaitu jumlah 0 pada oktet terakhir yaitu 6, jadi 2⁶-2 = 62

Subnet terdiri dari 192.168.4.0, 192.168.4.64, 192.168.4.128, 192.168.4.192

maka dengan itu IP range (untuk host) dapat diketahui yaitu

192.168.4.1 - 192.168.4.62 dan 192.168.4.63 sbg broadcastnya subnet 192.168.4.0

26. Berikut pernyataan yang benar mengenai MASQUERADE adalah a. Suatu cara menghubungkan ip public menjadi ip private b. Membuat ip private c. Membuat ip public d. Mentranslasi ip public menjadi ip private e. Merubah ip menjadi biner​


Jawaban:

d. Mentranslasi IP public menjadi IP private.

Penjelasan:

MASQUERADE adalah teknik yang digunakan dalam jaringan komputer untuk menyembunyikan alamat IP asli dari host yang terhubung ke internet. Saat menggunakan MASQUERADE, alamat IP publik dari jaringan lokal akan diubah menjadi alamat IP privat sebelum data dikirim ke internet. Ini memungkinkan host dalam jaringan lokal untuk berkomunikasi dengan internet menggunakan alamat IP privat yang ditranslasikan oleh perangkat jaringan yang melaksanakan MASQUERADE.


27. Soal 1. Carilah IP di masing-masing komputer a. 150 komputer 6.30 komputer C. 10 komputer d. 2 komputer dengan IP AWAL : 200.172.10.10


Jawaban:

Untuk mencari alamat IP pada masing-masing komputer, bisa dilakukan dengan beberapa cara, antara lain:

Pada sistem operasi Windows, buka Command Prompt dan ketikkan "ipconfig" (tanpa tanda petik), kemudian tekan enter. Alamat IP akan terlihat pada bagian "IPv4 Address".

Pada sistem operasi Mac, buka Terminal dan ketikkan "ifconfig" (tanpa tanda petik), kemudian tekan enter. Alamat IP akan terlihat pada bagian "inet".

Setelah mengetahui alamat IP, Anda dapat menggunakan pola alamat IP yang sama dan menambahkan angka pada bagian terakhirnya untuk mencari alamat IP pada komputer lain. Misalnya, jika IP awal adalah 200.172.10.10, maka IP pada komputer berikutnya adalah 200.172.10.11, 200.172.10.12, dst.

Untuk komputer A (150 komputer), bisa ditemukan dengan menambahkan angka dari 0 hingga 149 pada bagian terakhir alamat IP.

Untuk komputer B (6.30 komputer), bisa ditemukan dengan menambahkan angka dari 0 hingga 29 pada bagian terakhir alamat IP.

Untuk komputer C (10 komputer), bisa ditemukan dengan menambahkan angka dari 0 hingga 9 pada bagian terakhir alamat IP.

Untuk komputer D (2 komputer), bisa ditemukan dengan menambahkan angka dari 0 hingga 1 pada bagian terakhir alamat IP.
