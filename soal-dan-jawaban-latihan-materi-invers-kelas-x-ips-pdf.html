19. mohon bantuan beserta caranya materi fungsi invers kelas X, tolong yang jawab yang serius aja ya... deadline jam 3 sore nanti :')​


Jawab:

Konsep invers :

[tex]y = f(x)[/tex]  ->  [tex]x = f^{-1}(y)[/tex]

dari konsep tersebut berarti pada soal ini kita ubah menjadi

[tex]f(x) = \frac{4x-7}{2x-1}[/tex]

[tex]y = \frac{4x-7}{2x-1}[/tex]    -> tahap selanjutnya kita kali silang

[tex](y)*(2x-1) = 4x-7[/tex]

[tex]2xy-y = 4x-7[/tex]    -> karena hasil invers nantinya seperti ini [tex]x = f^{-1}(y)[/tex] , maka kita pindahkan variable "y" ke sebelah kanan :

[tex]2xy-4x = y-7[/tex]    -> untuk memindahkan variable "y" kita perlu mengeluarkan variable "x" terlebih dahulu :

[tex]2x(y-2) = y-7[/tex]        

[tex]2x = \frac{y-7}{y-2}[/tex]  

[tex]x = \frac{y-7}{2(y-2)}[/tex]      

[tex]x = \frac{y-7}{2y-4}[/tex]   -> sudah terbentuk hasil inversnya [tex]x = f^{-1}(y)[/tex] tinggal kita ubah variable nya menjadi :

[tex]f^{-1} (y) = \frac{y-7}{2y-4}[/tex]

[tex]f^{-1} (x)= \frac{x-7}{2x-4}[/tex]

cara cepat invers fungsi model pecahan :

 [tex]f(x)= \frac{ax+b}{cx+d}[/tex]   ->    [tex]f^{-1} (x)= \frac{-dx+b}{cx+(-a)} = \frac{-dx+b}{cx-a}[/tex]

[tex]f(x) = \frac{4x-7}{2x-1}[/tex]    ->   [tex]f^{-1} (x)= \frac{-(-1)x-7}{2x+ (-4)}[/tex] =  [tex]f^{-1} (x)= \frac{x-7}{2x-4}[/tex]

23. 7 Soal Latihan Materi invers Fungsi Tentukanlah F-(x) dari fungsi-pungsi berikut 1. f (x) = 2x -4 2. F(x) = 2x + 4 7 X - 3 3. F(x) = x² + 8x + 1​


Jawab:Untuk menentukan F-(x) dari fungsi-fungsi tersebut, kita perlu mengambil turunan terhadap variabel x.

1. Untuk fungsi f(x) = 2x - 4:

  Turunan dari fungsi tersebut adalah: f'(x) = 2.

  Jadi, F-(x) dari fungsi tersebut adalah: F-(x) = 2.

2. Untuk fungsi F(x) = 2x + 4 - 7x^3:

3. Untuk fungsi F(x) = x^2 + 8x + 1:

  Turunan dari fungsi tersebut adalah: F'(x) = 2x + 8.

  Jadi, F-(x) dari fungsi tersebut adalah: F-(x) = 2x + 8.
