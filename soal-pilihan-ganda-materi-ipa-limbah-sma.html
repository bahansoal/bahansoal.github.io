Jawaban:

1. Integrasi data adalah proses menggabungkan data dari berbagai sumber menjadi satu kesatuan yang lebih lengkap dan terstruktur. Apa tujuan utama dari integrasi data?

b. Meningkatkan keamanan data

c. Meningkatkan kualitas data

d. Meningkatkan efisiensi pengolahan data

a. Data redundancy

b. Data inconsistency

c. Data heterogeneity

d. Data integrity

3. Metode integrasi data yang menggunakan aturan dan logika untuk menggabungkan data dari berbagai sumber disebut:

a. Data warehousing

b. Data mining

c. Data cleansing

d. Data matching

a. Data warehouse

b. Data mart

c. Data lake

d. Data repository

a. Data cleansing

b. Data transformation

c. Data aggregation

d. Data normalization

6. Salah satu manfaat dari integrasi data adalah meningkatkan kualitas data. Apa yang dimaksud dengan kualitas data?

a. Keakuratan dan ketepatan data

b. Jumlah data yang tersedia

c. Kecepatan akses data

d. Ukuran data yang besar

7. Salah satu metode dalam integrasi data adalah dengan menggunakan teknologi yang memungkinkan penggabungan data secara real-time. Apa istilah yang digunakan untuk metode ini?

a. ETL (Extract, Transform, Load)

b. ELT (Extract, Load, Transform)

c. EAI (Enterprise Application Integration)

d. ESB (Enterprise Service Bus)

8. Salah satu tantangan dalam integrasi data adalah adanya perbedaan dalam penggunaan istilah dan konvensi dalam sumber data yang berbeda. Apa istilah yang digunakan untuk menggambarkan perbedaan ini?

a. Data redundancy

b. Data inconsistency

c. Data heterogeneity

d. Data integrity

9. Salah satu pendekatan dalam integrasi data adalah dengan menggunakan teknologi yang memungkinkan akses data secara terdistribusi dan real-time. Apa istilah yang digunakan untuk pendekatan ini?

a. Data warehousing

b. Data federation

c. Data mining

d. Data cleansing

a. Kecepatan akses data

b. Keakuratan dan ketepatan data

c. Jumlah data yang tersedia
