Penjelasan dengan langkah-langkah:

60 x 4 = 240

70 x n = 70n

80 x 6 = 480

90 x 8 = 720

100 x 3 = 300

rata rata = 80

jumlah siswa = 21 + n

jumlah nilai = 1820 + 70n

rata rata = jumlah nilai / jumlah siswa

80 = 1820 + 70n / 21 + n

80(21 + n) = 1820 + 70n

1680 + 80n = 1820 + 70n

80n - 70n = 1820 - 1680

10n = 140

n= 14

jadi banyak siswa yang mendapat nilai 70 adalah 14 anak

4 + P = 40

36n = 248 - 33

36n = 216

n = 216 : 36
